# AB-Inteligence

## Artificial Brain Intelligence

### Giving `real` inteligence to AI

AB Inteligence is a French project aiming to enable an AI to simulate consciousness. 
The AI ​​can control the "E-Brain," a simulated brain in a digital space. 
While the AI ​​cannot (at least not yet) possess true consciousness, the more powerful a model is, 
the better it will be able to simulate it. 
Given that these are early versions, the results may seem strange,
because at this stage, it's like **giving a human body to an ant**.
